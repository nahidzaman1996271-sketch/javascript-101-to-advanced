# 🥋 JavaScript Dojo

My personal journey of learning JavaScript — from the basics to advanced concepts, one commit at a time.

This repository documents everything I learn: notes, practice exercises, mini-projects, and code snippets, organized as I progress from fundamentals to more advanced topics.

I'm a Software Engineering student, and this repo is where I track my JavaScript learning journey. The goal is to build strong fundamentals, stay consistent, and eventually use these skills in real-world web development projects.

- 🎓 Learning style: hands-on, code-first, with notes alongside
- 🗓️ Updated regularly as I progress through topics
- 🛠️ Includes exercises, small projects, and experiments
